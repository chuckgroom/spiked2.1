Release: Spiked 2.1

Sept. 18
Spiked 2.0 has been revised and Spiked 2.1 has been released. This update fixes many of the bugs in Spiked 2.0 and expands the Spiked game.

Bugs Fixed

  • Once in a while (about twice and hour) Spiked would freeze between matches.
  • During a match, the game would just up and quit for no particular reason.
  • If you didn't have a "Spiked Contstants" (now called "Spiked Physics") file, then the game would bug out.

New Features

  • Spiked now works in 256, thousands, and millions of colors.
  • Spiked features new background graphics for the prefs. and about screens. These screens are more clearly organized and let the user find information faster.
  • There is a new weapon, the Rockyspiker, which turns a rock into a rockyspike and vice-versa. Rockyspikes are like spikes, except that they can be destroyed by spikes.
  • The event processor is more optimized, so gives you snappier response times.
  • You can now pause in the middle of a match.
  • You can have multiples of the same item, e.g. five cannonballs, at once.
